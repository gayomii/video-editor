# 📹 Web Video Editor

비디오 파일을 불러와 간단한 비디오 편집(움짤 생성, 길이 조절) 이 가능한 웹 사이트

<br/>

### 기술 스택

---

사용 언어

- Javascipt (React.js)

사용 라이브러리

- ffmpeg
- video-react
- mui

기타

- figma
- github
- prettier
